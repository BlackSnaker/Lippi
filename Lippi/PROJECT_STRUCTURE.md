# Lippi Project Structure

## Core
- `Lippi/Core/Localization` - language environment and localization helpers.
- `Lippi/Core/Widgets` - shared keys/update glue for widgets.

## UI
- `Lippi/UI/DesignSystem.swift` - design system, glass components, shared styles, shared visual helpers.
- `Lippi/UI/AnimatedAppleBackdrop.swift` - reusable animated backdrop.

## Features
- `Lippi/Features/Alarm` - alarm-related feature code.
- `Lippi/Features/Countdown` - countdown card and related UI.
- `Lippi/Features/EyeHealth` - eye health game + eye health screens.
- `Lippi/Features/Pomodoro` - pomodoro screen.
- `Lippi/Features/Settings` - settings screen + settings subcomponents.
- `Lippi/Features/Stats` - statistics card and chart UI.
- `Lippi/Features/Tasks` - tasks list, task row, add/edit task screen.
- `Lippi/Features/Today` - today/home screen.

## App Shell
- `Lippi/ContentView.swift` - app shell/navigation + app-wide stores/models + app entry.
